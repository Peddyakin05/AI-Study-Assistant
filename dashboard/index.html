Install in 3 steps

Load the extension from the downloaded folder — no Chrome Web Store needed.

1
Download the extension
Download the extension/ folder from this package. This contains the manifest, content script, background worker, popup, and styles.
2
Load into Chrome
Open chrome://extensions, enable Developer mode (top-right toggle), click "Load unpacked", and select the extension/ folder.
3
Add your API key
Click the extension icon → Settings tab → paste your Anthropic API key. Without a key, the assistant uses heuristic fallback mode.
